[데이터 수집 여부]
• 예, 일부 데이터를 수집합니다.

----------------------------------------------------------------------
1) 기기 또는 기타 ID (Device or other IDs)
----------------------------------------------------------------------
수집 여부: 예
유형: 광고 ID (예: Google 광고 ID / GAID, OS가 허용하는 광고 식별자)

처리 목적 (Play 폼에서 선택 가능한 항목에 맞춰 체크):
• 광고 또는 마케팅
• 앱 기능 (사기 방지·남용 방지·광고 사기 방지 등이 해당되면 선택)
• 분석

데이터가 필수인지: 아니오 (앱 핵심 기능(워드 퍼즐)은 광고 ID 없이 동작 가능. 광고·측정에 사용.)

데이터 암호화: 전송 중 암호화됨 (HTTPS/TLS 등 Google 광고 인프라 관행에 따름)

사용자가 삭제를 요청할 수 있는지: 예 (기기 설정에서 광고 ID 재설정·삭제, 앱 내 UMP/개인정보 옵션, 당사 문의 등)

----------------------------------------------------------------------
2) 위치 (Location)
----------------------------------------------------------------------
정밀 위치: 수집 안 함 (앱이 GPS 등 정밀 위치 권한을 요청하지 않음)

대략적인 위치: “앱이 직접” 위치 권한으로 수집하지는 않으나, 아래와 같이 제3자 SDK를 통해 간접적으로 처리될 수 있음을 명시하는 것이 안전합니다.

수집 여부 (광고 SDK 경유): 예 (또는 “제3자가 수집할 수 있음”으로 기술하는 방식 검토)
설명용 문구 예시:
  “WordMap은 위치 권한을 요청하지 않습니다. Google AdMob 등 광고 SDK가
   IP 주소 등을 이용해 국가·광역 수준의 대략적인 위치를 파악할 수 있으며,
   이는 광고 게재·측정·규정 준수 목적에 사용될 수 있습니다.”

처리 목적: 광고 또는 마케팅, 분석 (해당 시)

----------------------------------------------------------------------
3) IP 주소
----------------------------------------------------------------------
(Play 폼에서 “IP 주소”가 별도 항목인 경우)
수집 여부: 예 (광고·분석·사기 방지 목적으로 네트워크 통신 시 처리)
제3자: Google AdMob (Google LLC)

----------------------------------------------------------------------
4) 앱 활동 / 진단 / 기타
----------------------------------------------------------------------
AdMob·Google Play 서비스가 수집할 수 있는 기술·진단 정보(크래시, 성능, OS 버전 등)는
Google의 데이터 보안 공개 정책에 따릅니다. Play 데이터 보안 양식에서
해당 카테고리를 사용하는 경우, “제3자 SDK(AdMob)를 통해 수집될 수 있음”으로 정리합니다.

----------------------------------------------------------------------
[데이터 공유]
----------------------------------------------------------------------
• 예, 사용자 데이터를 제3자와 공유합니다.
공유 대상 예: Google LLC (Google Mobile Ads SDK / AdMob)
목적: 맞춤형 광고, 비맞춤형 광고, 광고 측정, 사기 방지 등

----------------------------------------------------------------------
[보안 관행]
----------------------------------------------------------------------
• 데이터 전송 시 암호화 (업계 표준 관행, Google 광고 네트워크 정책에 따름)

----------------------------------------------------------------------
[데이터 삭제 요청]
----------------------------------------------------------------------
• 예 (또는 정책에 맞는 옵션): 이용자가 삭제·동의 철회를 요청할 수 있는 경로를
  개인정보 처리방침에 명시 (기기 광고 ID 재설정, UMP 재호출, 운영자 이메일 등).

================================================================================
영문 요약 (Play 영어 양식용 메모)
================================================================================
- Device IDs: Collected via Google AdMob for ads, analytics, fraud prevention. Not required for core gameplay.
- Precise location: Not collected by the app (no GPS permission).
- Approximate location: May be derived by ad partners (e.g., from IP) for advertising/measurement.
- IP address: Processed for ads/analytics as part of ad serving.
- Shared with third parties: Yes — Google (AdMob) per their policies.
